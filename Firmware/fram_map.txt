DEFINED BY 2WIRELESS.INO (firmware)
0x000000 - 0x02BD18 Scratch area for preset and firmware transfers (179481 bytes)

0x02BD19 - 0x02BD39 32 byte SSID (33 locations allocated to accommodate null termination)
0x02BD3A - 0x02BD7A 63 byte WPA password (8 bytes minumum)
0x02BD7B - 0x02BD7D Optional three byte user page starting address (please use space above 0x02C000).
0x02BD7E - 0x02BD7F Optional two byte user page length (defaults to 0 to use default homepage).

DEFINED BY PRESETMANAGER.HTML (default homepage scratch area) 0x02BD80 - 0x2BFFF
0x02BD80 - 0x02BD85 "BUCHLA" identifier used for PresetManager.html initialization.
0x02BD86            Current preset. Two digit preset number ex. "01"
0x02BD87 - 0x2BFFD  Thirty 21 character preset names.

UNDEFINED
0x02C000 - 0x03FFFF Free and unused by default. WriteUserPage.html places custom user page here by default.
